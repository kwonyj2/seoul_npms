NMS 자동 등록 흐름
  1. 자산 관리(/assets/)에서 장비 등록 또는 변환
  2. 장비 상세 → 네트워크 설정 탭에서
    관리 IP · SNMP Community 입력 후 저장
  3. 저장 즉시 NMS 모니터링 장비에 자동 추가
  4. Celery가 SNMP 폴링하여 상태(정상/장애) 자동 갱신
